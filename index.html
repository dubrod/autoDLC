    Automatic Dynamic Lightbox Carousel. Upload your photos to a folder. Insert 1 Div. Link the JS & CSS and Done.

    DEMO

    Why autoDLC

    1. Quick Setup
    2. CSS REsizing not JS
    3. Dynamic HTML Build
    4. Responsive Carousel
    5. Total CSS Control
    6. Preloading optional

    Quick Setup in the .JS

    thumbFolder = "tns/";

    photosFolder = "photos/";

    magnifierImg = "images/zoom-in.png";

    prevArrow = "&#xe018;";

    nextArrow = "&#xe015;";

    Upload your Photos

    Thumbnails and Large Photos must be named the same.

    Easy Placement

    Place this div where you want the thumbnails to be. CSS will resize the thumb containers automatically.

    <div id="autoDLC"></div>
